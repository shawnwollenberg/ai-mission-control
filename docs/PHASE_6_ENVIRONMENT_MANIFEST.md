# Phase 6 Production Environment Manifest

All values are production-specific. `production:validate -- <process>` fails readiness when a required value is absent and never prints values.

Agent credentials generated after launch are stored as verifiers in PostgreSQL and displayed once to the human. Initial schedules live in durable projections rather than environment variables; timezone, missed-run policy, and concurrency policy are configured per schedule.
